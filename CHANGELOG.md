## [1.1.0] - 2026-03-16

### Изменено
- **UX**: Переработана система запроса разрешений. Теперь при старте приложения запрашивается только базовое местоположение (Location). Разрешения для работы в фоновом режиме (Background Location) запрашиваются отдельно, когда пользователь осознанно нажимает кнопку старта записи во вкладке сканирования, чтобы предотвратить агрессивную блокировку приложения ОС Android.
- **UI**: В `MainActivity` добавлена заглушка (Empty State), которая отображается вместо бесконечных диалогов с ошибкой, если пользователь отказался выдать права на местоположение при старте.

### Исправлено
- **Местоположение**: Выполнен стабильный переход на `FusedLocationProvider` для корректного сканирования Wi-Fi при слабом GPS-сигнала.
